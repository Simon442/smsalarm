SmsAlarm changelog

v2.1(version code 4) 
-Möjligt att kvittera ett inkommande primärlarm till ett telefonnummer
-Kvitteringen återuppringer om upptaget vid kvittering
-Fel loggas till fil
-Förbättrat hanteringen av datalagring och hämtning av data
-Förbättrat gränssnitt, mera anpassat för Android version 4.0 och uppåt
-SmsAlarm kan lyssna på flera telefonnummer än ett för sekundärlarm
-Ny ikon i notifikationsfältet för sekundärlarm
-Notifikationern skriver inte över varandra
-Möjlighet att spela upp larmsignalen två gånger vid larm
-Möjlighet att inaktivera SmsAlarm helt och hållet
-Möjlighet att ange sin räddningstjänsts namn
-Nya ringsignaler för sökare Swissphone Quattro 86, Swissphone Quattro 98, Swissphone Quattro XLS,
 Swissphone Patron Pro, Swissphone DE 900, Motorola Pageboy 2000, Motorola Firestorm, Motorola BMD,
 Bosch FME 87, Bosch FME 88, AEG TSE80 samt några diverse sökarljud. Totalt 67 nya sökarljud
-Applikationen följer enhetens ljudinställningar på ett mycket stabilare sätt

v1.1.2(version code 3)
-Gjort om about dialogen
-Lokaliserat appen till engelska, svenska och finska
-Lagt till stöd för kompensering av flera landskoder(svenska, norska, danska, tyska)

v1.1(version code 1)
-Sms Alarm reagerar på två numer(primär- och sekundär).
-Det är möjligt att ställa in appen så den tar hänsyn till telefonens ljudinställningar på ett korrekt sätt.
-Bättre hantering av mottagna sms.
-Förbättrat gränssnitt.
-SmsReceivern har fått ökad prioritet
