SmsAlarm changelog

v2.2(version code 9)
-Möjligt att lägga till fler än ett telefonnummer för primärlarm
-Möjligt att ange alfanumeriska tecken för telefonnummer
-Möjligt att ställa in fritexttriggning både för primär- och sekundärlarm
-Widgeten har fått ett till fält där den triggande texten syns, widgetenhar dessutom fått en annan formatering av kvitteringstiden
-Alarmloggen har utökats för att även visa det triggande ordet

v2.1.4(version code 8)
-Lade till hantering för ytterligare landskoder(USA, Canada, Slovenia, Österrike, Nya Zeeland och Storbrittanien)
-Bättre versionshantering, behöver bara ändra versionsnumret i manifest och i default string resursen

v2.1.3(version code 7)
-Patch, fixade bug som ledde till att applikationen stoppas om SD kortet inte är tillgängligt för skrivning.

v2.1.2(version code 6)
-Patch, fixade bug som ledde till att applikationen tvångsavslutas om man avbryter kvittering och sedan ringer ett samtal

v2.1.1(version code 5)
-Patch, ökade tiden för hur länge ett samtal måste vara utan att återuppringning ska ske

v2.1(version code 4) 
-Möjligt att kvittera ett inkommande primärlarm till ett telefonnummer
-Kvitteringen återuppringer om upptaget vid kvittering, beta stadie ännu
-Fel loggas till fil
-Förbättrat hanteringen av datalagring och hämtning av data
-Förbättrat gränssnitt, mera anpassat för Android version 4.0 och uppåt
-SmsAlarm kan lyssna på flera telefonnummer än ett för sekundärlarm
-Ny ikon i notifikationsfältet för sekundärlarm, grått kryss
-Notifikationern skriver inte över varandra
-Möjlighet att spela upp larmsignalen två gånger vid larm
-Möjlighet att inaktivera SmsAlarm helt och hållet
-Möjlighet att ange sin räddningstjänsts namn
-Nytt vibrationsmönster
-Nya ringsignaler för sökare Swissphone Quattro 86, Swissphone Quattro 98, Swissphone Quattro XLS,
 Swissphone Patron Pro, Swissphone DE 900, Motorola Pageboy 2000, Motorola Firestorm, Motorola BMD,
 Bosch FME 87, Bosch FME 88, AEG TSE80, Virve samt några diverse sökarljud. Totalt 67 nya sökarljud
-Applikationen följer enhetens ljudinställningar på ett mycket stabilare sätt
-Skärmen tänds vid inkommande larm(om den är mörk), och är tänd i 20 sekunder innan den släcks igen
-Lysdioden blinkar rött vid inkommande larm, när skärmen släcks, detta beror helt förstås också på om enheten stöder detta
-Kontakt information finns i "Om Sms Alarm" dialogen
-Inkommande larm(sms som applikationen reagerar på) loggas till en html fil
-Widget har lagts till med vilken man kan toggla om sms alarm ska vara aktiverat och om enheten ska använda sig av enhetens ljudinställningar.
Dessutom visas senast mottagna larm(primär och sekundär). Man kan även komma direkt till Sms Alarm härifrån och även komma till alarmlog-filen.
-Sms Alarm finns nu på svenska, engelska, tyska och finska.

v1.1.2(version code 3)
-Gjort om about dialogen
-Lokaliserat appen till engelska, svenska och finska
-Lagt till stöd för kompensering av flera landskoder(svenska, norska, danska, tyska)

v1.1(version code 1)
-Sms Alarm reagerar på två numer(primär- och sekundär).
-Det är möjligt att ställa in appen så den tar hänsyn till telefonens ljudinställningar på ett korrekt sätt.
-Bättre hantering av mottagna sms.
-Förbättrat gränssnitt.
-SmsReceivern har fått ökad prioritet
